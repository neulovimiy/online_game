Описание:
Это многопользовательская онлайн-игра, в которой игроки управляют машинками и взаимодействуют друг с другом в реальном времени.
Основные возможности:
Онлайн мультиплеер (несколько игроков одновременно)
Реальное время (движение машин синхронизируется между игроками)
Простая игровая физика
Клиент-серверная архитектура
Технологии:
Frontend: HTML, CSS, JavaScript
Backend: Node.js (сервер)
Networking: WebSocket / Socket.IO (уточни что у тебя)
Работа с реальным временем
Что я реализовал:
Логику движения машин
Синхронизацию игроков через сервер
Обработку пользовательского ввода
Архитектуру клиент-серверного взаимодействия
Как запустить:
git clone https://github.com/neulovimiy/online_game.git
cd online_game
npm install
npm start
Цель проекта:
Проект был создан в рамках обучения для изучения:
сетевого взаимодействия
real-time приложений
разработки игр в браузере
