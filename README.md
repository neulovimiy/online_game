# Многопользовательская онлайн-игра

Многопользовательская онлайн-игра, в которой игроки управляют машинками и взаимодействуют друг с другом в реальном времени.

---

## Основные возможности

- **Онлайн мультиплеер** — несколько игроков одновременно
- **Реальное время** — движение машин синхронизируется между игроками
- **Простая игровая физика**
- **Клиент-серверная архитектура**

---

## Технологии

| Компонент | Технологии |
|-----------|------------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Node.js |
| Networking | WebSocket / Socket.IO |
| Взаимодействие | Real-time |

---

## Что реализовано

- Логика движения машин
- Синхронизация игроков через сервер
- Обработка пользовательского ввода
- Архитектура клиент-серверного взаимодействия

---

## Цель проекта

Проект создан в рамках обучения для изучения:

- Сетевого взаимодействия
- Real-time приложений
- Разработки игр в браузере
